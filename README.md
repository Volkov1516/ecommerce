# ECOMMERCE

## Description
A clone of the Comfy. The main goal of the project is to practice `Full-Stack` development.

## Features
- Complex UI layout with custom components
- Product filtering
- Product sorting
- Pagination
- User registration and authentication
- Shopping cart functionality
- Product comments and rating

## Features
- **Front-End:** HTML, CSS, SaaS, Typescript, Next, Axios
- **Back-End:** Node, Express, Mongo, Mongoose
- **Sequrity:** bcrypt, cors, dotenv, jsonwebtoken

## Installing
- Clone the repository
- Install dependencies for client and server `npm install`
- Setup enviroment variables
- Run the server `npm run dev`
- Run the client `npm run dev`
